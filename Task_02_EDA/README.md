Task-02: Exploratory Data Analysis (EDA)

Objective:
To perform data cleaning and exploratory data analysis on the Titanic dataset
to identify patterns and relationships between variables.

Dataset:
Titanic Dataset from Kaggle

Tools Used:
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

Steps Performed:
- Data loading and inspection
- Handling missing values
- Univariate and bivariate analysis
- Data visualization

Conclusion:
Key insights about survival patterns were identified based on gender, class, and age.
